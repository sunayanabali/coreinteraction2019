Text Shadows

What is text shadow?

CSS3 text shadows can be applied to any text on the page. The syntax for text shadows is as follows:

text-shadow:xPosition yPosition blurSize color;

Where:
xPosition is the horizontal position of the text shadow relative to the text. yPosition is the vertical position of the text shadow relative to the text. blurSize is the size of the shadows blur. color is the color value â€“ hex, rgb, rgba, hsl, hsla or named color.
